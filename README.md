# 📊 Excel Analytics Portfolio
Welcome to my Excel Analytics Portfolio!

This repository is a collection of projects I built when I first started learning Excel for data analysis. Driven by the belief that data is the new oil, I created these projects to build a strong foundation in data manipulation, visualization, and interpretation. This portfolio documents my early journey in learning how to transform messy, raw datasets into clean, actionable business insights.

## 📂 Projects Included
This repository contains several distinct analytical projects, each focusing on different industries and datasets.

### 1. 👗 [Clothing Store Analysis](./clothing-store-analysis/)
An analysis of sales, orders, and customer demographics for a retail clothing store.

- **Highlights:** Tracks orders vs. sales by month, sales by gender, order status, and top performing states.

### 2. ☕ [Coffee Sales Analysis](./coffee-sales-analysis/)
A deep dive into coffee shop orders, exploring customer purchasing habits and product performance.

- **Highlights:** Connects multiple data tables (orders, customers, products) to generate comprehensive pivot tables and an interactive dashboard.

### 3. 🛒 [E-commerce Profitability Analysis](./ecommerce-profitability-analysis/)
An evaluation of e-commerce performance focusing on profit margins, sales volume, and regional growth.

- **Highlights:** Features Year-over-Year (YoY) growth tracking, profitability by category and subcategory, and state-level sales distribution.

### 4. 🌸 [Ferns and Petals Sales Analysis](./ferns-and-petals-sales-analysis/)
A commercial analysis for a floral and gifting business, investigating customer behaviors and top-selling products.

- **Highlights:** Utilizes customer, order, and product datasets to build high-level KPIs and business performance visualizations.

### 5. 🎬 [IMDb Movie Analysis](./imdb-movie-analysis/)
An exploratory data analysis of the top movies of all time using IMDb data.

- **Highlights:** Showcases the ability to filter, sort, and visualize entertainment industry data to find trends among critically acclaimed films.

### 6. 🏅 [Olympics Performance Analysis](./olympics-performance-analysis/)
A historical overview of Olympic medals and athletic performance across different countries and events.

- **Highlights:** Transforms raw historical data into a structured format to visualize global athletic success metrics.

## 🛠️ Skills & Techniques Demonstrated
Across these initial learning projects, I have applied and developed the following core Excel skills:

- **Power Query (ETL):** Using Power Query to automate the extraction, transformation, and loading of raw datasets.
- **Data Cleaning & Transformation:** Converting raw `.csv` and `.xlsx` files into processed, ready-to-analyze formats.
- **Data Modeling:** Connecting different tables (e.g., customers, products, orders) to create a cohesive relational model.
- **Pivot Tables:** Aggregating large datasets to extract meaningful summaries and KPIs.
- **Interactive Dashboards:** Building visual reports using charts, graphs, and slicers to allow end-users to dynamically filter insights.
- **Business Intelligence:** Translating data into practical business metrics like YoY growth, profit margins, and demographic breakdowns.
